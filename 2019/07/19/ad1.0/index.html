一、广告的基本知识

1.广告的目的

广告和搜索、推荐是有区别的
eg：对于广告，文字比图片点击高；对于推荐，图片比文字点击高，思考~（可从链路终止与传递的角度看）

什么是广告

完成较低成本的用户接触
（相较于人员性的信息传播，有效到达reach）

品牌广告brand awareness：提升较长时间的离线转化率
效果广告direct response：短期内明确的转化用户行为诉求
品效合一

2.广告的有效性模型

阶段——主要原则

  • 选择
    • 曝光
      • 取决于广告位的天然属性，优化空间小或0
    • 关注：技术可以优化
      • 不要干扰、打断用户的任务
      • 明确揭示推荐的原因
      • 符合用户的兴趣和需求
  • 解释
    • 理解
      • 广告在用户能理解的兴趣范围内
      • 与关注程度相匹配的理解门槛（电视和网络可放的广告时间与用户存留时间匹配）
    • 信息接受
      • 广告商认可度
      • 广告位、媒体认可度
  • 态度
    • 保持
      • 艺术性带来的记忆效果
    • 购买
      • 在用户价格敏感的接受范围内

效果广告比较关注：点击率、转化率
在各个阶段，越前对点击率影响越大，越后对转化率影响越大

时代广告每天reach几十万，并不多，但是因为它可以让用户觉得广告主可信度高，增加信息接受的阶段

广告策略的效果（有利有弊）——传统&在线

增加了关注，但是让很多人对品牌有质疑，美国广告控制擦边球
折扣——不再完全等同其他手段，是广告和营销之间的手段，自己本身增加了成本

传统：幽默&性感&艺术&折扣
线上：仿背景&大标识&简单

3.广告与销售的区别

广告：潜在用户
销售：明确需求者

精准广告不等于营销/CRM，也不简单是“把你的客户卖给你”“把竞争对手的客户卖给你”
display AD&search AD=提升各个环节转化&ROI高

SEM和SEO是包含关系
搜索引擎营销

4.在线广告特点

4.1在线广告独特性

  • 技术和计算导向
  • 可衡量性（但是从98年到现在，banner点击率从10%~0.1%，所以不能简单从点击率去看广告能力，只能在同一时间相对意义去比较）
  • 标准化
  • 媒体概念差异化

4.2在线广告协会

  • IAB：目的就是把线下广告预算搬到线上，会员主要是媒体方如谷歌facebook等~
  • 4A：主要是做【代理】的标准
    • 4A收取广告主的钱（收钱就会多考虑他们的利益）
    • （中国的代理公司都是不收广告主的钱，而是收取媒体的反点，因此不能代表广告主的利益，这是中国代理的问题）
  • ANA：主要代表广告主的利益（do not track）

6.计算广告核心问题和挑战

6.1在线广告的核心计算问题

对于品牌广告和效果广告——量+质
品牌：更看重量
效果：更看重质。但是必须基于量

u：用户
c：媒体
a：广告主
优化一组的ROI（ROI和具体产品形态有关）

  • 优化角度
    • 特征提取：受众定向，把u&c打上标签
    • 微观优化：CTR优化
    • 宏观优化：竞价市场机制
    • 受限优化：在线分配（对于质和量的协调）
    • 强化学习：探索（试的过程）与利用（优化收入）
    • 个性化重定向：推荐技术
  • 系统角度
    • 候选查询：实时索引
    • 特征储存：No-sql技术
    • 离线学习：Hadoop
    • 在线学习：流计算
    • 交易市场：实时竞价

6.2在线广告的核心主要挑战

  • 大规模
  • 动态性
  • 丰富的查询信息
  • 探索与发现

7.广告搜索推荐的比较

downstream：推荐在优化整个流的总收益，但是广告是一次就为终点——推荐和广告主要区别

8.投资回报率ROI优化

点击率的函数是：广告、用户、上下文（a、u、c）
点击价值：商品、用户本身（a、u）

在线广告ROI

  • Investment:#X×CPX
  • Return（每一次的求和）:点击率（CTR）×点击价值=eCPM

不同的分解对应不同的市场

  • CPM市场：eCPM（对supply有利）
  • CPC市场：动态CTR、固定click value（supply优化CTR，demand优化click value）
  • CPA/CPS/ROI市场：动态CTR+click value（对demand有利）
    • 问题
      • 没办法预估到达广告主click value
      • 广告主造假
      • 广告主骗展示（不会有人买，不用付钱，但是占领了用户心智）

CPM主要是品牌广告
CPC主要是效果广告
CPA/CPS/ROI要注意各种陷阱

9.在线广告系统结构

搭建一个广告系统有以下几个重要组成部分：

  1. 高并发的投送系统，即在线的AdServer，它通过请求中的user和context信息来决定展示哪些ads。它的特点是高并发，要做到10ms级别的实时决策，百亿次/天的广告投放系统。
  2. 受众定向平台，它进行离线的数据挖掘分析，一般要用到Hadoop。比如进行点击率预测的分析。
  3. 数据高速公路，它是联系在线与离线的部分，它比较简单，可以用开源工具实现，它的作用是准实时地将日志推送到其它平台上，目的一是快速地反馈到线上系统中，二是给BI人员快速看结果。它还可能收集其它平台的日志，比如搜索广告会收集搜索日志。
  4. 流式计算平台，它是比Hadoop快的一个准实时分析平台，它要实现的功能：反作弊，计价，实时索引。

三、受众定向

1.受众定向概念

1.中国互联网极其粗暴，各种强制跳出的广告。类比旅游景点强制拉客

用户的兴趣点不会因为网页拆分多个广告位而产生多倍的兴趣，所以通过广告位增加更多提升收益天花板低
